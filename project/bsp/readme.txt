LCD_T096P12 80*160 显示需转换成横屏
  the pins use as follow:
  - cs        <--->   pb12(software cs, pb12 as a general io to control flash cs)
  - sck       <--->   pb13
  - miso      <--->   pb14
  - mosi      <--->   pb15
  - usart1_tx <--->   pa9

  for more detailed information. please refer to the application note document AN0102.
